# 新闻与案例数据存储

本目录由后端与管理员页读写，请勿手动删除或移动。

## 目录结构

```
news-data/
├── latest-updates/     # 最新动态（新闻）
│   └── 2026-01-20_发布-JustPai-OS-2.0系统/
│       ├── meta.json   # 标题、日期、作者、摘要、正文块、封面路径、是否上架等
│       ├── cover.*    # 封面图
│       └── images/    # 正文内图片（可选）
└── case-studies/      # 合作案例
    └── 2025-10-15_全球知名车企亚太总部办公空间升级/
        ├── meta.json
        ├── cover.*
        └── images/
```

每条推送对应一个以「日期_标题」命名的文件夹，内部 `meta.json` 存文字与配置，图片存同目录或 `images/`。
