# JustPai 管理后台

用于发布、编辑、下架与删除「最新动态」和「合作案例」，数据通过后端 API 写入 `news-data` 文件夹。

## 启动

确保后端已启动（默认 http://localhost:3001），然后：

```bash
cd admin-page
npm install
npm run dev
```

管理后台默认运行在 http://localhost:5174，通过 Vite 代理将 `/api` 转发到后端。

## 功能

- **登录页**：输入账号密码后进入管理布局。
- **菜单**：最新动态、合作案例；右侧为列表与编辑区。
- **列表**：搜索、新增、编辑、下架（仅后台可见，前台不展示）、上架、删除（两次确认）。
- **新增/编辑**：标题、日期、作者、分类、摘要、正文（段落/小标题/图片块）、封面图；**推送**为上架并保存，**暂存**为仅保存不上架。

## 构建

```bash
npm run build
```

将 `dist` 部署到任意静态服务器，并配置将 `/api` 代理到后端地址。
